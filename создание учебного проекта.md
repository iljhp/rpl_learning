## Что нужно подключить для использования RPL

### 1. Заголовочные файлы
```cpp
#include <rpl/rpl.h>  // Основной заголовок (включает все)
```

### 2. CMake
```cmake
# Добавить lib_rpl
add_subdirectory(path/to/lib_rpl lib_rpl)

# Подключить к таргету
target_link_libraries(your_target
    PRIVATE
        desktop-app::lib_rpl
)
```

### 3. Зависимости
- C++17 или выше
- `desktop-app::external_gsl` (GSL — Guidelines Support Library)
